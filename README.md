# Executive Search Helper

https://github.com/user-attachments/assets/4df4dc32-3c7c-467e-bca1-054bb9a475b5

Executive Search Helper — macOS-приложение для executive search и рекрутмента.

Приложение помогает быстрее перейти от описания роли к рабочему поиску кандидатов: структурирует бриф, помогает сформировать поисковую логику, запускает поиск по открытым источникам и показывает найденных кандидатов в одном проекте.

Внутри проекта можно хранить описание роли, регионы поиска, найденных кандидатов, ссылки на профили, статусы и заметки. Для каждого поиска приложение показывает контекст: почему кандидаты появились, насколько они близки к задаче и что можно уточнить дальше.

Подходит для executive search консультантов, рекрутеров и HR-команд, которым нужно быстрее собирать первичный long list и держать работу по поиску в одном месте.

## Системные требования

macOS 14.6 или новее.

## Установка

1. Скачайте `ExecutiveSearchHelper-1.0.2-3.dmg` из [последнего релиза](https://github.com/SLMK14/executive-search-helper-download/releases/latest).
2. Откройте DMG.
3. Следуйте инструкции в открывшемся окне установки.

## Важно

При первом запуске macOS может показать предупреждение безопасности.

После установки откройте «Системные настройки» → «Конфиденциальность и безопасность» и нажмите «Открыть всё равно».
